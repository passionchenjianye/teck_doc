## 数据集表设计

#### 1. 数据集表

| 字段名称     | 字段类型    | 长度    |  是否必须  |  字段描述  |
| --------   | -------   | --------  | --------- |--------- |
| id     | bigint    | 32    |  是  |  主键ID  |
| name     | varchar    | 255    |  是  |  数据集名称  |
| type     | int    | 4    |  是  |  数据集类别  |
| create_time     | timestamp    |     |  是  |  创建时间  |
| creator     | varchar    | 255    |  是  |  创建人  |
